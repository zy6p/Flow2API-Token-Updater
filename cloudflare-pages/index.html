Firefox / Zen / Chrome / Edge 安装入口

这个页面提供扩展下载、安装说明和隐私政策。Firefox / Zen 用户优先使用 XPI； Chromium 用户使用 ZIP 后在 chrome://extensions 中加载已解压扩展。

下载

当前页面已包含可直接分发的安装包。

下载 Firefox / Zen XPI 下载 Chromium ZIP

备用文件： Firefox ZIP

查看隐私政策

安装说明

  • Firefox / Zen：优先尝试直接安装 .xpi。
  • 如果 Firefox / Zen 拒绝未签名扩展：打开 about:debugging#/runtime/this-firefox，临时加载 manifest.json。
  • Chrome / Edge：下载 ZIP 后解压，在扩展页面开启开发者模式并加载已解压扩展。
