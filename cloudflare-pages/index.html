安装说明

  • Firefox / Zen：下载“临时加载包”并解压。
  • 打开 about:debugging#/runtime/this-firefox，点击 Load Temporary Add-on，选择解压目录中的 manifest.json。
  • 未签名的 .xpi 在 Firefox / Zen 正式版中通常会被显示为“附加组件似乎已损坏”。
  • Chrome / Edge：下载 ZIP 后解压，在扩展页面开启开发者模式并加载已解压扩展。
  • 想证明自己下载的是最新包：先看页面顶部的版本号是否为 __FLOW2API_VERSION__。
  • 下载后解压扩展，检查其中 manifest.json 的 version 字段是否同样为 __FLOW2API_VERSION__。
  • 如果需要可验证文件完整性，对照 SHA256SUMS 或读取 latest.json 中记录的文件名与 SHA-256。
