Firefox / Zen 安装建议

  • 长期使用、多 profile 部署：安装签名自动更新版，后续更新由浏览器自动拉取。
  • 调试开发：使用临时加载包，在 about:debugging#/runtime/this-firefox 手动加载。
  • 签名自动更新版的内置更新地址：__FLOW2API_GECKO_SELFHOST_UPDATES_URL__
  • 如果浏览器提示站点访问权限，请直接允许“所有网站”，后续不用为每个 Flow2API 域名单独授权。

安装说明

  • Firefox / Zen 自动更新版：下载 latest-firefox-selfhost.xpi，通过文件安装或拖入浏览器扩展页。
  • Firefox / Zen 临时加载包：打开 about:debugging#/runtime/this-firefox，点击 Load Temporary Add-on，选择解压目录中的 manifest.json。
  • Chromium：解压 ZIP，在 chrome://extensions 开启开发者模式，点击“加载已解压的扩展程序”。
  • 如果需要可验证文件完整性，对照 SHA256SUMS 或读取 latest.json / updates.json 中记录的版本与 SHA-256。

发布链

  • AMO listed：用于公开上架与 AMO 用户更新。
  • AMO unlisted：用于签名自分发 XPI，供 Zen / Firefox 多 profile 长期安装。
  • Cloudflare Pages：托管下载页、latest.json、updates.json 和自分发 XPI。
  • Firefox 自更新依赖内置 update_url 指向 HTTPS 更新清单。
